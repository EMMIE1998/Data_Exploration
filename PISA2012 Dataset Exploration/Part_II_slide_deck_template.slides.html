Investigation Overview¶

For the presentation, I focus on just the influence of Wealth,ICT Resources on Read Plausible Value Mean and Math Plausible Value Mean of the PISA2012 data and leave out most of the intermediate derivations.

I start by introducing the Math Plausible Value Mean and Read Plausible Value Mean, followed by ICT Resources Variable and Wealth.

Then later, I have plotted a colored Scatterplot of Math Plausible Value Mean, Read Plausible Value Mean and Wealth and ICT Resources. I've made sure to use different color palettes for each quality variable to make sure it is clear that they're different between plots.

Dataset Overview¶

The Pisa Dataset has 485490 students and 636 features that describes each of them.

Features in the dataset that support my investigation.¶

This PISA2012 dataset has 636 columns, but we will only look at a few varibles that will help us answer our questions;

  1. ICT resources.
  2. Wealth
  3. Immigration Status
  4. Math Plausible Value Mean
  5. Read Plausible Value Mean
  6. Science Plausible Value Mean
